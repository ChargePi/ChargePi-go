# WS2811 and WS2812b

## Description

The WS281x LED strip comes in multiple voltage variants. It is recommended to use the 5V variant, since the Raspberry Pi
provides the 5V for power instead of using an external power supply.

## Wiring

|   RPI PIN    | WS281x PIN |  
|:------------:|:----------:|
|  Any 5V pin  |    VCC     | 
| Any GND pin  |    GND     |   
| 32 (GPIO 12) |    Data    |
